# Security Policy

Milestone 0 is pre-release software and does not yet have a stable API.

Report vulnerabilities privately using GitHub's private vulnerability reporting feature. Do not
open a public issue containing exploit details or personal data. Include affected versions,
reproduction steps, impact, and any suggested mitigation.

Maintainers will acknowledge a report within seven days and coordinate disclosure after a fix is
available. Supported code is the latest commit on the default branch.
